# 企业官网模板

这是一个可直接打开的静态官网模板，适合工程建设、洁净空间、设备安装、技术服务类公司做展示站初稿。

## 文件说明

- `index.html`：页面结构
- `styles.css`：视觉样式与响应式布局
- `script.js`：公司资料和页面内容配置

## 你现在要改的地方

优先修改 `script.js` 顶部的 `siteContent` 对象：

1. `brandName` 和 `brandTagline`
2. `heroTitle` 和 `heroDescription`
3. `services` 里的服务内容
4. `cases` 里的行业案例
5. `contact` 里的电话、邮箱、地址

## 使用方式

直接双击 `index.html` 就能预览。

如果后面你要继续做成正式版，下一步通常是：

1. 替换为你们公司的真实文案
2. 补充项目实拍图和资质图
3. 增加案例详情页和新闻详情页
4. 绑定正式域名并完成备案上线
